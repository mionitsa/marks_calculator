RUS:

Программа "Калькулятор оценок"(marks_calculator.py) позволяет рассчитать оценки, которые должен получить пользователь для достижения требумого балла.
Программа получает на вход строку оценок пользователя и требуемый балл.

Формат ввода:
Пользователь вводит оценки через пробел, а требуемый балл является целым или действительным числом.

Пример работы программы:
Ваши оценки: 3 4 5 4 3 4 4 4 5
Требуемый балл: 4.3

Программа выводит нужные оценки и их количества для достижения требуемого балла.

Ошибки:
Если ввод данных некорректен, то программа указывает ошибку.
Если требуемый балл меньше, чем средний балл, который имеет пользователь, то программы указывает на это.
Также, программы выводит ошибку, если требуемый балл недостижим.

ENG:

The program "Marks calculator" (marks_calculator.py ) allows you to calculate the scores that the user must receive in order to achieve the required score.
The program receives a string of user ratings and the required score as input.

Input format:
The user enters the scores separated by a space, and the required score is an integer or a real number.

Example of the program:
Your grades: 3 4 5 4 3 4 4 4 5
Required score: 4.3

The program outputs the necessary grades and their numbers to achieve the required score.

Mistakes:
If the data input is incorrect, the program indicates an error.
If the required score is less than the average score that the user has, then the program indicates this.
Also, the program outputs an error if the required score is not achievable.
